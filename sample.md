# Welcome to MarkdownReader

A beautiful, minimal markdown reader for macOS.

## Features

- **Beautiful typography** with generous line spacing
- *Italic text* for emphasis
- `Inline code` rendering
- ***Bold and italic*** combined

## Typography

Good typography is invisible. Great typography is *felt*. This reader is designed to make reading markdown files a pleasure, with careful attention to:

- Line height and paragraph spacing
- Font sizing and weight hierarchy
- Comfortable reading width (max ~760px)
- System font integration (SF Pro)

~~Strikethrough text~~ is also supported.

That's it — just open a `.md` file and enjoy reading.
