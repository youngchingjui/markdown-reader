# Inspiration

A running collection of things that caught my eye.

## Design

- **Stripe Press** — the way they treat long-form content as a first-class design surface. Every page feels like a book.
- **Linear changelogs** — proof that even release notes can be beautiful if you care enough.
- **Apple's Human Interface Guidelines** — the writing is as good as the design. Clear, opinionated, useful.

## Typography

- *The Elements of Typographic Style* by Robert Bringhurst — the bible. Return to it every few months.
- Butterick's Practical Typography — web-native, opinionated, free. The chapter on line spacing changed how I set type.

## Tools

- **iA Writer** — the benchmark for distraction-free writing. The monospace font choice is genius.
- **Figma** — still the best for collaborative UI work, despite the bloat.
- **Bear** — beautiful note-taking, markdown-native, fast search.
